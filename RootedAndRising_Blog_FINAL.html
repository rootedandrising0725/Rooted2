One-time database setup — takes 3 minutes
1

Go to firebase.google.com → click Get Started → sign in with Google → click Create a project → name it rooted-and-rising → disable Analytics → click Create project.

2

In the left sidebar click Firestore Database → Create database → choose Start in test mode → pick any location → click Done.

3

Click the ⚙️ gear → Project settings → scroll to Your apps → click the </> icon → name it blog → click Register app. Copy the values from the config block and paste them below.
